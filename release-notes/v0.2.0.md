# What's New in Version 0.2.0

**Released:** 2026-04-21

---

## ✨ New Features

### Generation Progress Tracking
When you click **Generate TIP**, you no longer get redirected away to the drafts list. Instead, an inline progress panel appears directly on the Generate page showing:
- Which batch Claude is currently processing (e.g. "Processing batch 3 of 13…")
- A live progress bar with percentage
- Token count when available
- Automatic navigation to your draft the moment generation completes

---

## 🐛 Bug Fixes

### Full Discovery Workbook Data Now Included
VM inventory, IP address tables, VLAN configurations, and firewall rules from your Excel discovery workbooks were previously being silently truncated after ~3 pages. All data is now passed in full to Claude and rendered as properly formatted tables in the generated TIP.

### Numbered Lists Reset Per Section
Numbered lists in exported Word documents previously ran continuously across the entire document. They now correctly restart at 1 at the beginning of each section.

### Export Formatting Improvements
Exported `.docx` files now use proper Thrive brand formatting:
- Calibri 10.5pt body font
- Zebra-striped table rows
- Consistent cell padding and vertical alignment
- Template header and footer inherited correctly
- Document Control Notice stripped from output

### Bullet and Numbered List Rendering
Bullets and numbered lists in exports now use proper Word XML formatting (real list styles, not plain hyphens), matching how Word renders native lists.

---

## 💡 What You Need to Know

- Generation still takes 2–4 minutes for large discovery workbooks — stay on the Generate page to watch progress
- Existing drafts are unaffected
- No action required on your end
