# What's New in Version 0.8.0

**Released:** 2026-04-23

---

## ✨ New Features

### Document Type Management
Admins and users can now change the type of any uploaded document directly from the Documents page — no need to re-upload if a file was auto-classified incorrectly.

**What you can do:**
- Change a document between Discovery Worksheet, Service Order, and Other
- Updates take effect immediately and are reflected everywhere the document appears

**How to use it:**
1. Go to **Documents**
2. Find the document you want to update
3. Click the type badge (e.g. "Discovery Worksheet") — it becomes a dropdown
4. Select the correct type

---

### TIP Library — Edit Title & Category (Admin)
Administrators can now edit the title and category of any Library document directly inline — no need to delete and re-upload to correct a name or recategorize a TIP.

**What you can do:**
- Rename any library document
- Change its category (with autocomplete suggestions from existing categories)
- Works on pending, approved, and rejected documents

**How to use it:**
1. Go to **Library**
2. Find the document to edit
3. Click the **pencil icon** on the right side of the row
4. Edit the title and/or category in the inline fields
5. Click **✓** to save or **✗** to cancel

---

## 💡 What You Need to Know

- Document type changes are instant — no page refresh needed
- Library edits are admin-only; regular users can still view all library documents
- Category edits will clear the AI-suggested flag, marking the category as manually confirmed
