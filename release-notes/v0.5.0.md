# What's New in Version 0.5.0

**Released:** April 22, 2026

---

## ✨ New Features

### Bring Your Own Claude API Key (BYOK)

You can now connect your own Anthropic Claude API key to the TIP Generator. Generation uses your key directly — no shared system key.

**What you can do:**
- Add your personal Anthropic API key in Profile Settings
- Remove or replace your key at any time
- Generation is blocked with a clear message if no key is set

**How to use it:**
1. Click your name in the top-right header (or the key icon)
2. Navigate to **Profile Settings**
3. Paste your API key from [console.anthropic.com](https://console.anthropic.com)
4. Click **Save**

---

### Choose Your Claude Model

After adding your API key, the app automatically fetches the current list of available Claude models directly from Anthropic — always up to date.

**What you can do:**
- Select which Claude model is used for all TIP generation and AI Assist
- Refresh the model list anytime to see newly released models
- If no model is selected, the app defaults to claude-sonnet-4-5

**How to use it:**
1. Go to **Profile Settings**
2. Once your API key is saved, the **Claude Model** section appears
3. Select your preferred model from the live list
4. Your choice is saved automatically

---

## 💡 What You Need to Know

- **Generation requires an API key.** If you haven't added one, clicking Generate will show a clear error with a link to your Profile Settings.
- **Your key is never displayed** after saving — the profile page only shows whether a key is set.
- **Model choice persists** across sessions — you only need to set it once.
