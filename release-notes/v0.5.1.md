# What's New in Version 0.5.1

**Released:** April 22, 2026

---

## ✨ New Features

### TIP Library Transparency

When a TIP is generated using library examples, you can now see exactly which examples were used — right in the draft editor.

**What you'll see:**
- A purple **"X library examples"** badge in the draft header (hover over it to see the document titles)
- A small **book icon** next to every section header, indicating the content was influenced by library examples

**Why this matters:**
- You'll know at a glance whether library context was applied to your generation
- Helpful for understanding why certain sections match a particular structure or style

---

## 💡 What You Need to Know

- This information is stored permanently — reloading the page will still show which examples were used
- Only drafts generated after this update will show library indicators; older drafts will not
- No action required to use this feature — it works automatically when approved library documents are available
