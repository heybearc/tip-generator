# What's New in Version 0.4.0

**Released:** 2026-04-22

---

## ✨ New Features

### Draft Duplication
You can now duplicate any existing draft with a single click. The copy button (📋) appears on every draft row in the Drafts list and creates an identical draft at the top of the list — useful when you want to try a different edit direction without losing your original work.

**How to use it:**
1. Go to the **Drafts** page
2. Click the **copy icon** on any draft row
3. A duplicate draft appears at the top of the list, ready to edit

---

### Gap Report (Missing Data Finder)
Every draft now has a built-in gap report that scans for `[DATA NEEDED: ...]` placeholders — sections the AI couldn't populate because information was missing from your source documents.

**How to use it:**
1. Open any draft
2. Click the **Gaps** button in the draft header
3. An amber panel expands showing a numbered list of every data gap
4. Use the list to know exactly what information you still need to gather before delivery

**What you'll see:**
- Count of gaps at the top
- Each gap listed with the field name and description
- "No gaps found" if all placeholders were resolved

---

## 💡 What You Need to Know

- Duplicated drafts are independent copies — editing one does not affect the other
- The gap report is read-only and updates every time you open it
- Gaps only appear for placeholders inserted by the AI; manually added `[DATA NEEDED:]` text will also be detected
